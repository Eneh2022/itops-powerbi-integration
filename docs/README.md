# ITOps Power BI Integration

## Overview
This project focuses on integrating IT Operations data (helpdesk, servers, user activity) into Power BI dashboards for real-time insights and decision-making.

## Objectives
- Centralize IT data
- Enable real-time monitoring
- Improve decision-making

## Tools Used
- Power BI
- SharePoint
- Power Automate
- Microsoft 365

## Current Phase
Phase 1: Helpdesk ticket logging system (SharePoint)

## Future Plans
- Asset lifecycle tracking
- Predictive analytics
